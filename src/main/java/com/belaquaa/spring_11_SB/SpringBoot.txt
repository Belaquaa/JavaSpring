Spring Boot — это модуль Spring, который упрощает процесс создания и разработки Spring-приложений за счёт следующих
своих возможностей:
 - Автоконфигурация: Spring Boot автоматически настраивает ваше приложение на основе добавленных в проект зависимостей.
   Это означает, что если, например, в вашем пути к классам есть библиотека для работы с базой данных, Spring Boot
   автоматически настроит соединение с базой данных.
 - Стартеры: Стартеры — это набор удобных зависимостей-агрегаторов, которые включают в себя все необходимые библиотеки
   для работы с определенной технологией. Стартеры упрощают процесс добавления компонентов в ваш проект, так как вам
   не нужно искать и добавлять каждую библиотеку вручную, а так же следить за совместимостью их версий.
 - Встроенный сервер: Spring Boot включает в себя встроенный сервер (обычно Tomcat, но можно использовать и другие
   серверы, например Jetty или Undertow), что позволяет запускать веб-приложения без необходимости внешнего сервера
   приложений.
