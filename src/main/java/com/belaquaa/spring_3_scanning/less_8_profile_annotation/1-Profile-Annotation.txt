Аннотация @Profile в Spring Framework используется для указания, что определенный компонент (класс, метод,
конфигурация) должен быть активирован только в определенном профиле. Профили в Spring позволяют разработчикам
регистрировать различные компоненты в зависимости от окружения, в котором запущено приложение, например, разработка,
тестирование, стейджинг или продакшн.

Профили могут быть активированы различными способами, включая параметры среды выполнения, системные свойства, файлы
конфигурации и другие.
